# ⚔️ 钭骊渊大战AI — Minecraft Fight

> 一款运行在浏览器中的我的世界风格 2D 格斗游戏。  
> 单文件、零依赖，打开即玩。

---

## 🎮 游戏简介

《钭骊渊大战AI》是一款像素风 2D 格斗游戏。玩家扮演 **钭骊渊**，与 AI 对手在我的世界风格的竞技场中展开决斗。游戏共 **5 个难度关卡**，每关采用 **三局两胜制**，击败全部关卡即为最终胜利。

---

## 🕹️ 操作说明

### ⌨️ 键盘（PC）

| 按键 | 动作 |
|------|------|
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `W` / `空格` / `↑` | 跳跃 |
| `J` | 拳击 |
| `K` | 剑击 |
| `L` | 特殊技（范围爆炸） |
| `U` | 发射附魔弓箭（需拾取道具） |

| 区域 | 动作 |
|------|------|
| 左侧上半 | 跳跃 |
| 左侧左区 | 向左移动 |
| 左侧右区 | 向右移动 |
| 右侧顶部 | 附魔弓（GUN） |
| 右侧上区 | 特殊技（SPEC） |
| 右侧中区 | 剑击（SWORD） |
| 右侧下区 | 拳击（PUNCH） |

## ⚔️ 攻击技能

| 技能 | 伤害 | 冷却 | 说明 |
|------|------|------|------|
| 🥊 拳击 | 8 | 20帧 | 快速近身攻击 |
| 🗡️ 剑击 | 15 | 45帧 | 中等伤害，较大攻击范围 |

## 📈 关卡难度

| 关卡 | 名称 | AI决策间隔 | AI伤害倍率 |
|------|------|-----------|-----------|
| 第一关 | 简单 | 15–25帧 | ×0.6 |
| 第二关 | 普通 | 12–20帧 | ×0.75 |
| 第三关 | 困难 | 8–12帧 | ×1.0 |
| 第四关 | 极难 | 5–10帧 | ×1.3 |
| 第五关 | 地狱 | 3–7帧 | ×1.6 |

每关采用**三局两胜**。全部5关通关后可重新挑战。

## 🎵 音效系统

游戏使用 Web Audio API 实时合成音效，无需任何音频文件：

- 🎼 **背景音乐**：C# 五声音阶琶音，战斗开始后淡入
- 🥊 **拳击音**：方波短促音
- 🗡️ **剑击音**：锯齿波滑音
- 💥 **特殊技**：白噪声爆炸
- 💀 **KO 音**：双音下行
- 🏆 **胜利音**：四音上行旋律
- 🔔 **回合开始**：三音提示音

> ⚠️ 首次点击或按键后音频自动解锁（浏览器安全策略限制）。

## 🚀 运行方式

直接用浏览器打开 `index.html` 即可，无需服务器、无需安装。

```bash
# 本地快速启动（可选）
python3 -m http.server 8080
# 然后访问 http://localhost:8080/minecraft-fight/
```

兼容所有现代浏览器（Chrome / Firefox / Safari / Edge），支持 PC 键盘和移动端触摸操作。

## 🎨 技术实现

- **渲染**：HTML5 Canvas 2D，像素风格渲染（`image-rendering: pixelated`）
- **音频**：Web Audio API 实时程序化合成
- **动画**：`requestAnimationFrame` 游戏循环，帧率无关物理
- **AI**：状态机决策（approach / attack / dodge / retreat），难度越高决策越快、越激进
- **特效**：粒子系统（方块碎片）、屏幕震动、KO爆炸动画
- **字体**：像素点阵自定义英文字符 + Canvas 原生中文渲染
